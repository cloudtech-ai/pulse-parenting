🔑 How to Get a YouTube Data API Key

✅ Step 1: Go to Google Cloud Console

Visit: https://console.cloud.google.com/

✅ Step 2: Create or Select a Project

  • At the top, click the project dropdown
  • Click “New Project” or select an existing one
  • Give it a name like “YouTube API Project”
  • Click “Create”

✅ Step 3: Enable YouTube Data API

  1. After selecting your project, go to the API Library:
    https://console.cloud.google.com/apis/library
  2. Search for: YouTube Data API v3
  3. Click on the result and click “Enable”

✅ Step 4: Create API Key

  1. Go to the Credentials page:
    https://console.cloud.google.com/apis/credentials
  2. Click “+ Create Credentials” → choose “API key”
  3. Google will generate an API key. Copy it

✅ Step 5: (Recommended) Restrict Your API Key

  1. On the API key screen, click “Restrict key”
  2. Under API restrictions, select “Restrict key” and choose YouTube Data API v3
  3. Under Application restrictions, choose the relevant restriction type (e.g. HTTP referrers for web apps)

✅ Step 6: Save and Use Your API Key

You can now use this API key in your app to make requests like:

https://www.googleapis.com/youtube/v3/search?part=snippet&q=DBT&key=YOUR_API_KEY

🧠 Notes

  • Quota: Free tier gives 10,000 units/day
  • Typical usage:
    • search.list: 100 units per call
    • videos.list: 1 unit per call
